Phase 1

Foundations (3–4 months)

Build the bedrock: math, programming, tools, and workflow.

Core Topics

  • Mathematics & Statistics: Linear algebra (vectors, matrices, eigenvalues), Calculus (derivatives, gradients), Probability (Bayes, distributions), Statistics (mean, variance, hypothesis testing, CIs)
  • Programming: Python (numpy, pandas, matplotlib, seaborn), R (optional), Git & GitHub
  • Tools: Jupyter/VS Code, virtual envs (conda, venv), Google Colab

Recommended Resources

  • Khan Academy (Math)
  • Think Stats — Allen B. Downey
  • Python for Data Analysis — Wes McKinney
  • MIT 6.0001 — Intro to CS in Python
Phase 2

Data Wrangling & Exploratory Data Analysis (2–3 months)

Turn raw data into usable insights and features.

What to Master

  • Data cleaning (missing values, outliers)
  • Feature engineering (binning, encoding, scaling)
  • EDA (distribution, correlation, profiling)

Tools & Libraries

pandas, numpy, matplotlib, seaborn, plotly, scipy

Project Ideas

  • Titanic survival EDA
  • Netflix ratings analysis
  • World Bank indicators dashboard
Phase 3

Machine Learning Fundamentals (3–4 months)

Classic algorithms, evaluation, and problem framing.

Algorithms

  • Supervised: Linear/Logistic Regression, Decision Trees, SVM, KNN
  • Unsupervised: K-Means, DBSCAN, PCA
  • Evaluation: CV, accuracy, precision/recall, F1, ROC-AUC

Libraries & Projects

scikit-learn, xgboost, lightgbm

  • Credit risk prediction
  • Customer segmentation
  • Digit classification (MNIST)
Phase 4

Deep Learning & Neural Networks (4–6 months)

Modern neural architectures and training techniques.

Concepts

  • NNs (forward/backprop), loss functions, optimizers (SGD, Adam), activations
  • CNNs, RNNs/LSTMs, Transformers

Stack & Projects

TensorFlow/Keras, PyTorch, Hugging Face Transformers

  • Image classification (CIFAR-10, chest X-rays)
  • Sentiment/spam text classification
  • Sequence modeling (forecasting, translation)
Phase 5

Specializations (4–6 months)

Go deep in NLP, Computer Vision, or RL.

NLP — tokenization, embeddings, Transformers
  • Text preprocessing (tokenization, lemmatization)
  • Embeddings: Word2Vec, GloVe
  • Transformers: BERT, GPT
Computer Vision — augmentation, detection, segmentation
  • Image preprocessing & augmentation
  • Object detection (YOLO, SSD)
  • Semantic/instance segmentation
Reinforcement Learning (optional)
  • MDPs, policies, value functions
  • Q-learning, DQN, PPO
Phase 6

MLOps & Deployment (2–3 months)

Ship reliable models with automation and monitoring.

Core Topics

  • APIs: Flask, FastAPI
  • CI/CD pipelines
  • Docker, Kubernetes
  • Monitoring & drift detection

Tools

MLflow, DVC, Weights & Biases, AWS/GCP/Azure

Phase 7

Capstone & Career Transition (3+ months)

Build a standout portfolio and ship real value.

Capstone Projects

  • Full-stack ML app with deployment
  • Real-world datasets with measurable impact
  • Research-style project with reproducible notebooks

Portfolio

  • GitHub repos with strong READMEs
  • Personal website/blog
  • Kaggle + certifications (Coursera, DeepLearning.AI, edX)
Bonus

Academic & Creative Stretch Goals

Elevate your craft and contribute to the community.

  • Read research papers (arXiv, Papers with Code)
  • Contribute to open source
  • Join hackathons & data clubs
  • Write your own paper and submit to a venue
